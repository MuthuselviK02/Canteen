📊 Dynamic Time Logic by Status

⏰ Pending Orders

  • Base Wait Time: 20 minutes (default)
  • Queue Delay: 5 minutes per position (User), 3 minutes per position (Kitchen)
  • Formula: Base Time + (Queue Position × Delay)
  • Display: "~25 min", "~35 min", etc.

🔄 Preparing Orders

  • Base Prep Time: 15 minutes (default)
  • Elapsed Time: Real-time calculation from started_preparation_at
  • Remaining: Base Prep Time - Elapsed Time
  • Display Logic:
    • ≤ 0 minutes: "Almost ready"
    • ≤ 2 minutes: "1 min", "2 min"
    • > 2 minutes: "~5 min", "~10 min"

✅ Ready Orders

  • User Display: "Ready for pickup"
  • Kitchen Display: "Ready"
  • Time Shown: Time when order became ready

🎉 Completed Orders

  • Display: "Completed"
  • Time Shown: Completion timestamp
  • No Estimated Time: N/A for completed orders
